Recent Work

Yoda Weather is a single page app utilizing the AngularJS framework. Weather can be boring so what better way to spruce up the weather than Yoda? Yoda Weather App relies on the Openweathermap API.

Current and five day forecasts are provided for any city in the world. Yoda also offers his take on the weather which will randomly vary depending the on weather conditions.

View app

Snackshack is a single page app which allows users to search for a type of food in specified city via the Foursquare API. Ratings, user reviews, venue address, and likes are provided for each venue.

View app

Halloween Quiz was built with JQuery and JS in the spirit of Halloween. Users answer multiple choice questions and receive feedback after each answer along with some fun facts. Spooky audio elements were implemented to add to the immersive experience.

View app

Countries & Capitals, built with AngularJS and the Geonames API, allows users to view details on any country in the world. A full list of countries are populated. When a user clicks a country they are taken to the details view for that country.

View app

Band Guru allows users to view detailed information on their favorite bands and musicians. Band Guru was created with JQuery, JS. The app relies on AJAX requests from the Echonest API.

View app

FlickSpire allows users to find inspirational images via the Flickr API. Users can search a term and view a gallery related to that topic. FlickSpire utilizes AJAX, JQuery, and JS.
